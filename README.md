# Splatter Paintball Adventure Hub

A modern and minimal landing page for university adventure activities. This project presents a clean one-screen experience with large action buttons that guide visitors to different adventure-themed pages.

## Overview

This website is designed for a campus event or activity showcase and includes:

- A polished landing page with a focused visual style
- Four large buttons for major activity choices
- Separate HTML pages for each adventure experience
- Responsive layout that works well on desktop and mobile screens

## Project Structure

```text
Splatter-Paintball/
├── index.html
├── Ambush/
│   ├── ambush-adventure.html
│   └── paintball-challenge.html
└── NightCamp/
    ├── jungle_trekking_night_camp.html
    └── night-camp.html
```

## Features

- Minimal and professional design
- Easy navigation between activity pages
- Smooth hover effects and modern card-style layout
- Simple structure that is easy to expand

## How to View the Project

You can preview the website locally in two simple ways:

1. Open [index.html](index.html) directly in your browser.
2. Or run a local server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000/
```

You can easily customize the project by:

- changing the text and headings in [index.html](index.html)
